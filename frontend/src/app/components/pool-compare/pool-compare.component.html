Pool Playground

Compare card pools across up to 4 investigators

Adding Investigators

Empty slot
Click ＋ and type a name to search. The full legal card pool loads automatically once selected.
Removing
Click the × on a filled slot. Supports 2–4 investigators simultaneously.

Set Algebra Modes

Any
Default. This investigator's pool contributes freely — cards appear regardless of what others can or cannot take.
Must ●
Restricts results to cards available to every Must investigator. Use to find the true intersection of pools.
Not ●
Subtracts this investigator's pool from results. Use to isolate cards the others can take that this one cannot.
Presets
Union — everyone on Any, all cards shown.
All share — everyone on Must, intersection only.
Only [Name] — one Must, all others Not — exclusive cards for that investigator.

Reading Results

Groups
Cards are bucketed by how many active investigators share access — from "Shared by all" down to individual exclusives. Click any group header to collapse it.
Presence avatars
Each row shows a portrait per investigator. Bright = can take this card. Faded = cannot.

Filters

XP
All / Level 0 / XP 1+. Applies globally across all groups.
Name & Text
Name searches the card title. Text searches inside card body and flavor text — useful for keyword hunting.
Type, Faction, Slot, Cost, Trait, Set
Multi-select dropdowns. Select multiple values to broaden within that filter. Active values appear as chips — click any chip to remove it.
Clear all
Resets every filter and mode back to Union defaults.

Examples

Cards Roland and Wendy both can run at Level 0
Add both → All share preset → Level 0 filter.
Assets only Skids has access to, that Finn cannot run
Skids → Must · Finn → Not · Type = Asset.
Seeker events shared by exactly 3 of 4 investigators
Add all four → Faction = Seeker, Type = Event → read the "Shared by 3" group.
